一年多前我其实就有搭建个人博客网站的念头，因为我平时很喜欢看博客。无论技术还是生活，看着他人笔下写就的文字总能带给我不同的感触，这也让我有了提笔记录的冲动。

但可惜的是，作为一个行动力与专注力双低的人，我已经经历了数次：**「寻找网站模板 -> 新建 Git 仓库 -> 稍作魔改 -> 嫌弃并放弃 -> 再次寻找网站模板...」** 的痛苦循环🤣。不过今天实在是太无聊了，想写点代码又不知道从何写起。那么...是时候终结这个循环了...（？突然中二）于是和 Cursor 经过一番通力合作，最终的效果如你所见。

希望以后可以写下一些有温度的文字，不止于技术，也包括生活。
